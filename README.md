# LuMe Clinic CRM - The Private Atelier

A high-end, bespoke Clinical Relationship Management (CRM) system designed for private medical practices and concierge clinics.

## Design Philosophy: "The Private Atelier"
LuMe Clinic moves away from the cluttered, line-heavy density of traditional CRMs toward a layout that breathes with intention.

- **Intentional Asymmetry:** Generous, unequal white space guides the eye toward critical patient data.
- **Tonal Depth:** A dark-mode-first approach where depth is defined by light and surface elevation, not by lines.
- **Editorial Scaling:** Dramatic contrasts between Manrope display typography and Inter metadata labels.

## Features
- **Executive Overview:** Real-time clinical vitals, revenue tracking, and occupancy metrics.
- **Patient Registry:** High-fidelity patient records with condition tracking and status monitoring.
- **Clinical Calendar:** Specialist schedule coordination and suite availability management.
- **System Activity:** Real-time audit trail of lab results, referrals, and billing alerts.
- **Digital Prescriptions:** Integrated script management and pharmacy hub synchronization.

## Tech Stack
- **Frontend:** React 18, Vite, Tailwind CSS v4, Motion (for animations).
- **Backend:** Node.js, Express (with Vite middleware integration).
- **Icons:** Lucide React.
- **Typography:** Manrope (Headlines), Inter (Body).

## Getting Started
1. Install dependencies: `npm install`
2. Start the development server: `npm run dev`
3. Build for production: `npm run build`

## Security & Privacy
Designed with HIPAA-compliant structures in mind, utilizing encrypted data paths and secure concierge chat interfaces.
